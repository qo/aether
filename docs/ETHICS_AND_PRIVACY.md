# Ethics And Privacy

Purpose: set boundaries for responsible RF sensing experiments.

## What Is Confirmed / What Is Unknown

- [Confirmed in docs] avoids identity, emotion, hidden collection, and medical claims.
- [Unknown / needs verification] Every real dataset must be reviewed for consent metadata before sharing.

## Boundaries

- Do not collect probe requests, preferred-network lists, or MAC-linked histories.
- Do not infer identity, emotion, medical condition, heartbeat, or HRV.
- Make session recording explicit.
- Keep source mode visible.
- Store consent and session metadata with datasets.
- Report uncertainty plainly.

## Dataset Handling

Use pseudonymous device IDs in UI and reports. Do not publish raw recordings that could reveal private environment details without review.
